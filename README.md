# DSPAutoSelectEjector
戴森球计划 电磁弹射器自动选择可弹射的轨道

前期电磁弹射器建得不多，戴森云很缺的情况下，经常发现之前选的轨道一会会后就倾角过大罢工了，需要手动调整目标轨道。
自动选择后，可以尽可能地提高弹射器利用率。

当弹射器无法弹射时，会自动选择目前可以弹射的轨道。
